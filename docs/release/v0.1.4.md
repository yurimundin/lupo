# Lupo v0.1.4

Release incremental para Windows com auditoria visível do cofre e polimento de
acessibilidade.

## Novidades

- Nova tela "Auditoria do cofre" acessível pelo ícone de escudo no header.
- Resumo local por severidade: Crítico, Atenção e Baixo.
- Lista de achados com descrição e recomendação para cada risco encontrado.
- A auditoria considera entradas fora da Lixeira e roda 100% no dispositivo.

## Correções e polimento

- Diálogos passam a fornecer descrição acessível de forma consistente.
- O diálogo de confirmação agora tem fallback acessível quando o chamador passa
  apenas o título.
- O diálogo de novo grupo na raiz agora também possui descrição.
- README e roadmap atualizados para refletir a auditoria visível como parte do
  MVP.
