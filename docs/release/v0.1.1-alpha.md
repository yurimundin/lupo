# Sec.Basis v0.1.1-alpha

Alpha incremental para testers conhecidos.

## Destaques

- Recuperacao guiada ao detectar `.kdbx.tmp` pendente depois de crash ou
  gravacao interrompida.
- Restauracao conservadora de `.kdbx.bak`: antes de substituir o cofre atual,
  o app preserva uma copia `before-restore`.
- Abertura normal nao e interrompida por `.bak` sozinho, porque esse arquivo
  e esperado apos salvamentos bem-sucedidos.
- Versao sincronizada para novos instaladores Windows `.msi` e `.exe`.

## Notas importantes

- Sem assinatura de codigo nesta alpha. O Windows SmartScreen pode exibir aviso
  ao abrir o instalador.
- Sem auto-update. Atualizacoes futuras devem ser baixadas manualmente.
- Mantenha backup externo dos cofres usados em teste.

## Smoke flow recomendado

1. Instalar em uma maquina Windows limpa ou perfil limpo.
2. Abrir o app.
3. Criar ou abrir um cofre `.kdbx`.
4. Criar uma entrada, salvar, fechar e reabrir.
5. Criar um arquivo `.kdbx.tmp` ao lado do cofre e confirmar que o aviso de
   recuperacao aparece.
6. Confirmar que "Abrir cofre atual" segue sem restaurar backup.
7. Confirmar que "Restaurar backup" preserva o cofre atual em
   `.before-restore-*.bak`.
