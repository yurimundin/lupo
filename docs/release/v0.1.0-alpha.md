## Notas importantes

- Sem assinatura de código nesta alpha. O Windows SmartScreen pode exibir aviso
  ao abrir o instalador.
- Sem auto-update. Atualizações futuras devem ser baixadas manualmente.
- Mantenha backup externo dos cofres usados em teste.

## Smoke flow recomendado

1. Instalar em uma máquina Windows limpa ou perfil limpo.
2. Abrir o app.
3. Criar um cofre `.kdbx`.
4. Criar uma entrada com senha gerada.
5. Salvar, fechar e reabrir o app.
6. Validar busca, lixeira, restauração e bloqueio automático.
