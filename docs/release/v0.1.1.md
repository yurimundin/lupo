# Sec.Basis v0.1.1

Release incremental para validação controlada no Windows.

## Destaques

- Recuperação guiada ao detectar `.kdbx.tmp` pendente depois de crash ou
  gravação interrompida.
- Restauração conservadora de `.kdbx.bak`: antes de substituir o cofre atual,
  o app preserva uma cópia `before-restore`.
- Abertura normal não é interrompida por `.bak` sozinho, porque esse arquivo
  é esperado após salvamentos bem-sucedidos.
- Versão sincronizada para novos instaladores Windows `.msi` e `.exe`.

## Notas importantes

- Sem assinatura de código nesta versão. O Windows SmartScreen pode exibir
  aviso ao abrir o instalador.
- Sem auto-update. Atualizações futuras devem ser baixadas manualmente.
- Mantenha backup externo dos cofres usados em teste.

## Smoke flow recomendado

1. Instalar em uma máquina Windows limpa ou perfil limpo.
2. Abrir o app.
3. Criar ou abrir um cofre `.kdbx`.
4. Criar uma entrada, salvar, fechar e reabrir.
5. Criar um arquivo `.kdbx.tmp` ao lado do cofre e confirmar que o aviso de
   recuperação aparece.
6. Confirmar que "Abrir cofre atual" segue sem restaurar backup.
7. Confirmar que "Restaurar backup" preserva o cofre atual em
   `.before-restore-*.bak`.
