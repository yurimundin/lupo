# Lupo v0.1.7

Release incremental para Windows focada em saúde estrutural do código, redução
de acoplamento e preparação da base para evoluções futuras com menor risco.

## Novidades

- Camada `src/services` criada para concentrar integrações com Tauri,
  clipboard, abertura de URLs externas e ações sensíveis.
- Fluxos de abrir, criar e desbloquear cofre extraídos para hooks dedicados.
- Controllers extraídos para command palette, lista de entradas e detalhe de
  entrada.
- Operações KDBX de entradas divididas por responsabilidade: criação,
  duplicação, favorito, movimentação e Lixeira.
- Documentação reorganizada: `AGENTS.md` e `CLAUDE.md` agora são índices curtos,
  com a memória completa arquivada em `docs/architecture/project-memory.md`.

## Correções e polimento

- Facade `stores/vault.ts` reduzida para store e seletores; helpers puros
  voltaram a ser importados diretamente de `src/lib`.
- Removidos placeholders e assets de scaffold que não eram usados.
- Mantido botão textual "Copiar senha" para copiar a senha sem revelar.
- Testes de ações sensíveis e bytes locais movidos para perto dos serviços que
  validam.
