# Lupo v0.1.3

Release incremental para Windows com foco em organização visual, refinamento de
UX e documentação.

## Novidades

- Pastas da sidebar agora aparecem em ordem alfabética, mantendo a Lixeira
  sempre no final.
- A personalização de pastas ganhou cores para os ícones.
- Nova opção de ícone "Jurídico", com balança Lucide.
- A tela "Sobre" ganhou uma chamada discreta para o Argo:
  trial de 30 dias sem cartão, com link para `https://argo.basis.app.br`.

## Correções e polimento

- Mantém os metadados visuais das pastas compatíveis com KeePass/KeePassXC.
- Adiciona descrições acessíveis aos diálogos de ícone, renomeação e
  recuperação de cofre.
- Atualiza README e roadmap com o estado atual do produto.
