# Lupo v0.1.5

Release incremental para Windows com polimento de arquitetura, histórico de
entradas mais completo e manutenção da base para próximas funcionalidades.

## Novidades

- Histórico de entradas mais útil: comparação entre versão atual e antiga,
  restauração, exclusão de versões antigas e retenção automática de até 20
  versões por entrada.
- Estrutura interna do app reorganizada para facilitar evolução segura:
  componentes de detalhe da entrada, anexos e sidebar de grupos foram separados
  em unidades menores.
- Criação de entradas movida para a camada KDBX, com rollback centralizado em
  caso de falha ao salvar.
- Testes KDBX divididos por domínio: core, entradas, histórico, anexos e ícones
  visuais de grupos.

## Correções e polimento

- Backend Tauri separado por responsabilidade (`commands/files`,
  `commands/recovery`, `commands/save` e `commands/crypto_commands`), mantendo a
  mesma API exposta ao frontend.
- Reuso de fluxos comuns para mutações do cofre e confirmação de descarte de
  mudanças não salvas.
- Remoção de selector legado de grupos que não era mais usado.
- README e roadmap atualizados com o estado atual de histórico, anexos, ícones,
  auditoria e compatibilidade Android planejada.
